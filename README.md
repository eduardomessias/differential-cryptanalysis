# Differential Cryptanalysis

## FEAL-4
This is a chosen-plaintext attack on FEAL-4 programmed in C as part of my studies in the discipline of Cryptography and Number Theory, during my masters in Blockchain and Decentralized Ledgers at Dublin City University, in 2022.
